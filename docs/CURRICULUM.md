# Complete A1 Curriculum Map

## Scope
- **80 lessons** across 16 weeks
- **Target:** Goethe-Zertifikat A1 pass-ready
- **Total active learning:** ~37 hours lessons + ~40 hours SRS/reviews/tests = ~77 hours
- **Narrative:** Jana's journey from Bratislava to Vienna

---

## Vocabulary Targets

| Category | Target words |
|---|---|
| Core vocabulary (Goethe A1 list) | 700 |
| Active vocabulary (can produce) | 400 |
| Passive vocabulary (can recognise) | 700 |
| New words per lesson average | 12–15 |
| SRS recycling ensures each word seen | minimum 5 times |

## Vocabulary Recycling Schedule

Every new word introduced in lesson X is scheduled to reappear in:
- X + 2 (verification)
- X + 5 (consolidation)
- X + 10 (long-term storage)
- X + 20 (mastery check)

This is managed by the SRS system automatically. Words which are answered incorrectly are recycled more aggressively.
