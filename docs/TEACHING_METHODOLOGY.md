# Teaching Methodology

## Core Approach: Communicative Language Teaching (CLT) + Spaced Repetition

This app combines four proven methodologies tailored for adult Slovak speakers learning German independently.

---

## Methodology 1: Communicative Language Teaching (CLT)

**Principle:** Language is learned by using it in real situations, not by memorising rules.

**How it is applied in the app:**
- Every lesson ends with a "communicative minimum" — one sentence a learner can say today to a real person
- Dialogue exercises simulate real conversations (at a café, asking directions, at the doctor)
- Grammar is introduced through example sentences first, rule explanation second
- The narrative character Jana provides a communicative context throughout all 80 lessons

**Why it works for Slovak speakers:**
Slovak has complex grammar but CLT reduces the cognitive load — learners use correct chunks before understanding all the rules.

---

## Methodology 2: Spaced Repetition System (SRS)

**Principle:** Review a word just before you forget it. Each successful recall increases the next review interval.

**Schedule:**
- New word → review after 1 day
- Correct recall → review after 3 days
- Correct again → 7 days → 14 days → 30 days
- Wrong answer → back to 1 day

**How it is applied:**
- Every vocabulary item from every lesson is tracked independently
- The SRS Review session (accessible from the home screen) queues due words
- Vocabulary items from the narrative (Jana's story) are prioritised as high-frequency

---

## Methodology 3: Input + Output Balance (Krashen + Swain)

**Krashen's Input Hypothesis:** Learners acquire language from comprehensible input slightly above their current level (i+1).

**Swain's Output Hypothesis:** Production (speaking/writing) forces noticing of gaps.

**Balance in app:**
| Lesson Phase | Type | Purpose |
|---|---|---|
| Flashcard | Input | First exposure |
| Listen | Input | Auditory form |
| Mini-text | Input | Contextual reading |
| MCQ | Input→Output | Recognition |
| Fill | Output | Controlled production |
| Free Write | Output | Free production |
| Dialogue | Input+Output | Communicative practice |

---

## Methodology 4: Habit Design for Procrastinators

Based on BJ Fogg's Tiny Habits and James Clear's Atomic Habits.

**Key design decisions:**
- **Lesson length max 25 min** — small enough to start at 10pm
- **Clear progress bar** — always know exactly where you are
- **Streak system** — don't break the chain
- **Each lesson has ONE clear goal** — "After this lesson I can ask for directions"
- **First 5 lessons are easiest** — early wins build confidence
- **No "lesson failed" state** — learners can retry any exercise, partial credit awarded
- **Micro-rewards** — XP, level-up animations, completion badges

**Character:** Jana Nováková, 28, from Bratislava
**Situation:** Jana gets a job offer in Vienna. The course follows her journey.

## Slovak-Specific Teaching Notes

| German feature | Slovak equivalent | Key difference |
|---|---|---|
| der/die/das gender | M/F/N gender | Gender assignment differs — must memorise |
| Accusative case | Slovak accusative | Direct object — similar concept |
| Dative case | Slovak dative | Slovak has richer case system — easier for Slovaks |
| du/Sie | ty/Vy | Nearly identical — use this connection |
| Verb-second word order | Slovak free order | German is STRICT — verb always position 2 |
| Satzklammer | No Slovak equivalent | Completely new concept — needs extra practice |
| Umlauts ä/ö/ü | No Slovak equivalent | Pronunciation must be drilled explicitly |
| ß vs ss | No Slovak equivalent | Rule: after long vowel/diphthong = ß |

## Goethe A1 Skill Weighting

Each lesson is tagged with which skill it primarily develops:

- 🟦 **Wortschatz** (Vocabulary)
- 🟩 **Grammatik** (Grammar)
- 🟥 **Hören** (Listening)
- 🟨 **Lesen** (Reading)
- 🟪 **Schreiben** (Writing)
- 🔶 **Sprechen** (Speaking)

Most lessons are multi-skill. Every lesson L70–L80 is all-skills exam preparation.
