## Grammar Notes

### kein declension table

```
           Mask.     Neut.     Fem.      Plural
Nom:       kein      kein      keine     keine
Akk:       keinen    kein      keine     keine
```

**Rule**: kein follows the SAME pattern as ein/einen. Only the **masculine accusative** adds -en.
- ❌ Ich habe kein Hunger. → ✅ Ich habe **keinen** Hunger. (der Hunger = M → Akk = keinen)
- ✅ Ich habe keine Zeit. (die Zeit = F → same in Nom and Akk)
- ✅ Ich habe kein Buch. (das Buch = N → same in Nom and Akk)
- ✅ Das ist kein Problem. (das Problem = N → Nominative)

**nicht vs. kein**: 
- `nicht` negates verbs or adjectives: Ich spreche *nicht* Russisch.
- `kein` replaces `ein/eine` or no article before nouns: Ich habe *keinen* Kaffee. (replaces *einen*)

## Teacher Notes
- Students often over-use `nicht` where `kein` is needed. Drill the substitution rule: wherever `ein` could go, use `kein` (not `nicht`).
- The masculine accusative `keinen` is the ONLY form change — emphasize this strongly.
- kein Problem! is a high-frequency phrase Austrians use constantly — introduce it as a chunk.
- The apartment-viewing dialogue gives authentic context: these exact phrases come up in real apartment search situations in Vienna (particularly on willhaben.at).
