## Grammar Notes

### es gibt + Accusative

**es gibt** = "there is / there are" — ALWAYS followed by accusative case!

```
es gibt + einen (M) / eine (F) / ein (N) / keine (Neg.)
```

Examples:
- Es gibt **einen** Aufzug. (der Aufzug = M → Akk → einen)
- Es gibt **eine** Küche. (die Küche = F → Akk → eine, same)
- Es gibt **ein** Badezimmer. (das Badezimmer = N → Akk → ein, same)
- Es gibt **keinen** Keller. (der Keller = M → Akk + neg → keinen)
- Es gibt **keine** Heizung. (die Heizung = F → Akk + neg → keine)

**Contrast with Slovak**: Slovak uses "je tu" + NOMINATIVE. German uses "es gibt" + ACCUSATIVE. This is a critical difference!
- SK: Je tu *výťah*. (nominatív) 
- DE: Es gibt *einen Aufzug*. (akuzatív)

### Floor numbering (Austrian context)
- Erdgeschoss (EG) = prízemie = ground floor
- 1. Stock / 1. Etage = 1. poschodie = 1st floor
- 2. Stock / 2. Etage = 2. poschodie = 2nd floor

- Austrian floor numbering differs from Slovak: Erdgeschoss ≠ prvé poschodie. It IS the ground floor.
- Real estate vocabulary matters: Jana needs these words to actually find an apartment in Vienna via willhaben.at.
- der Lift (Austrian) vs. der Fahrstuhl (German German) — introduce the Austrian variant since Jana is in Vienna.
