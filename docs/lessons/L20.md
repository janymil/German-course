## Grammar Notes

### How Separable Verbs Work

A separable verb has a **prefix** (Vorsilbe) that **detaches** in main clauses:

**In main clauses** → prefix SEPARATES and goes to the END:
```
[Prefix goes here ↓]        [Rest of sentence]     [Prefix ↑]
Der Zug  fährt   um 8 Uhr   ab.
Jana     steht   um 7 Uhr   auf.
Ich      kaufe   im Supermarkt  ein.
```

**General rule**: In a main clause, conjugated verb stays in position 2, prefix goes to the very end — just like the Satzklammer with modal verbs!

**In infinitive constructions** (with modal verbs) → prefix STAYS attached:
- Ich muss um 7 Uhr **aufstehen**. ✅ (infinitive = stays together)
- Ich muss **auf** um 7 Uhr **stehen**. ❌ (WRONG!)
- Jana möchte ihre Mutter **anrufen**. ✅

### Common separable prefixes and their meanings
| Prefix | Meaning tendency | Example |
|--------|-----------------|---------|
| **ab-** | away, departure | abfahren (depart), abholen (pick up) |
| **an-** | towards, beginning | ankommen (arrive), anrufen (call) |
| **auf-** | up, opening | aufstehen (get up), aufmachen (open) |
| **aus-** | out | ausgehen (go out), aussteigen (get out) |
| **ein-** | in, entry | einschlafen (fall asleep), einkaufen (shop) |
| **zu-** | closing | zumachen (close), zuhören (listen) |

### Recognition tip
In a dictionary or vocabulary list, separable verbs are written with a **pipe**: auf|stehen, ein|kaufen

### einladen — stem change reminder
- er **lädt** ein (a→ä stem change from L14!)
- Ich lade dich ein. / Er **lädt** mich ein.

## Teacher Notes
- The separable prefix rule is actually CONSISTENT — it always goes last. Connect this to the Satzklammer pattern from L16.
- "With modal verbs, the infinitive goes last — and when an infinitive is separable, it stays together." This is the key insight.
- die Bim (Viennese slang for Straßenbahn) — mention as local color, Jana will hear it in Vienna constantly.
- The minitext (SMS) format is very motivating — it's real-world, informal, and uses all the week 4 grammar.
- einladen combines L14 (stem change) + L20 (separable) — explicitly note this compound difficulty.
